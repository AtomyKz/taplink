# Atomy Taplink — Регистрационная страница

Красивый taplink для Instagram с формой регистрации в Atomy и интеграцией Google Sheets.

## 📁 Структура проекта

```
Atomy taplink/
├── index.html              # Главная страница
├── style.css               # Стили
├── script.js               # Логика (форма, калькулятор)
├── google-apps-script.js   # Код для Google Apps Script
└── README.md               # Инструкция
```

## 🚀 Быстрый старт

### 1. Настройка Google Sheets (обязательно!)

#### Шаг 1.1: Создайте Google Таблицу
1. Перейдите на [sheets.google.com](https://sheets.google.com)
2. Создайте новую таблицу
3. Переименуйте первый лист (например, на `Данные`)

#### Шаг 1.2: Настройте Apps Script
1. В меню таблицы выберите: **Расширения → Apps Script**
2. Удалите весь код по умолчанию
3. Вставьте весь код из файла `google-apps-script.js`
4. Сохраните проект (`Ctrl+S` / `Cmd+S`)

#### Шаг 1.3: Разверните веб-приложение
1. Нажмите **Развернуть** → **Новое развертывание**
2. Тип: **Веб-приложение**
3. **Кто имеет доступ:** выберите **"Все"** (Anyone)
4. Нажмите **Развернуть**
5. Подтвердите разрешения (если спросит)
6. Скопируйте **URL веб-приложения**

#### Шаг 1.4: Настройте авто-создание листов
1. В Apps Script слева нажмите на иконку **Триггеры** (часы ⏰)
2. Нажмите **+ Добавить триггер**
3. Выберите функцию: `createDailySheet`
4. Выберите источник событий: **По времени**
5. Тип таймера: **Дневной таймер**
6. Время: **с полуночи до 1:00**
7. Нажмите **Сохранить**

> Теперь каждый новый день будет автоматически создаваться новый лист с названием `дд.мм.гггг`!

### 2. Подключите URL к сайту

1. Откройте файл `script.js`
2. Найдите строку:
   ```javascript
   const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/ВАШ_ID/exec';
   ```
3. Замените `ВАШ_ID` на реальный ID из URL, который вы скопировали

### 3. Разместите на GitHub Pages (бесплатно!)

#### Шаг 3.1: Создайте репозиторий на GitHub
1. Перейдите на [github.com](https://github.com)
2. Нажмите **New** → создайте публичный репозиторий (название любое)

#### Шаг 3.2: Загрузите файлы
```bash
# В папке проекта выполните:
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ_НИК/НАЗВАНИЕ_РЕПО.git
git push -u origin main
```

ИЛИ просто загрузите файлы через веб-интерфейс GitHub.

#### Шаг 3.3: Включите GitHub Pages
1. На странице репозитория перейдите в **Settings**
2. Слева выберите **Pages**
3. В разделе **Source** выберите **Deploy from a branch**
4. Выберите ветку **main** и папку **/(root)**
5. Нажмите **Save**
6. Через 1-2 минуты сайт будет доступен по ссылке вида:
   ```
   https://ВАШ_НИК.github.io/НАЗВАНИЕ_РЕПО/
   ```

### 4. Добавьте ссылку в Instagram

1. Откройте Instagram
2. Перейдите в **Редактировать профиль**
3. В поле **Веб-сайт** вставьте URL вашего GitHub Pages
4. Сохраните

## 🔧 Кастомизация

### Изменить имя спонсора
Откройте `index.html`, найдите:
```html
<p class="sponsor-name">Елена Коханова</p>
<p class="sponsor-rank">Auto-sales Master</p>
```

### Изменить цвета
Откройте `style.css`, найдите `:root` и измените переменные:
```css
:root {
    --primary: #00ADEF;  /* Главный цвет */
    --cyan: #22d3ee;     /* Акцентный */
    --bg: #F8FAFC;       /* Фон */
}
```

### Изменить валюты/расчёты
Откройте `script.js`, найдите:
```javascript
const CURRENCIES = {
    KZT: { symbol: '₸', min: 10000, max: 100000, step: 1000, defaultOrder: 32000 },
    ...
};
```

## ❓ Частые вопросы

**Q: Данные не приходят в Google Sheets?**
A: Проверьте, что:
1. URL в `script.js` правильный
2. В Apps Script доступ выставлен на "Все"
3. Таблица не закрыта приватно (доступ "Редактирование" для вас)

**Q: Новый лист не создаётся?**
A: Проверьте триггер в Apps Script — он должен быть настроен на функцию `createDailySheet`

**Q: Сайт не открывается?**
A: Подождите 2-3 минуты после включения GitHub Pages. Проверьте, что репозиторий публичный.
